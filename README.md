# Rservex

`Rservex` is an under-development client for [Rserve](https://www.rforge.net/Rserve/index.html). 
Aiming to enable the R <--> Elixir interoperation.

## Current State

Currentle `Rservex` can:

 - Create a connection
 - Close a connection
 - Send a command without response
